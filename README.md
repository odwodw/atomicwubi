

# AtomicWubi - 原子五笔输入法练习平台

一个优雅的五笔输入法练习与学习 Web 应用，帮助用户提升五笔打字速度和准确率。

## 项目简介

AtomicWubi（原子五笔）是一款专为五笔输入法学习者设计的练习平台，采用现代化的界面设计和游戏化的学习机制，让五笔练习变得更加有趣和高效。

## 功能特性

### 🎯 核心功能

- **多模式练习**
  - 字符练习模式：通过不同难度等级的字符进行专项训练
  - 文章练习模式：输入整篇文章，边打字边学习
  - 词组练习：巩固五笔词组输入技能

- **学习进度追踪**
  - 每日经验值统计，记录每日学习情况
  - 已掌握字统计，了解个人学习成果
  - 连续天数统计，培养每日练习习惯

- **游戏化机制**
  - 成就系统：达成特定里程碑获得成就
  - 积分系统：根据正确率和速度计算得分
  - 连击系统：连续正确输入积累连击数

### ⚙️ 设置功能

- **音效开关**：开启/关闭打字音效反馈
- **盲打模式**：隐藏字根提示，考验真实水平
- **个人设置**：记住用户偏好设置

### 📊 数据统计

- 总学习天数
- 总输入经验值
- 已掌握字符数量
- 历史成绩查询

## 技术栈

- **前端框架**：原生 JavaScript + HTML5 + CSS3
- **数据存储**：LocalStorage 本地存储
- **音效系统**：HTML5 Audio API

## 使用说明

### 快速开始

1. 打开 `index.html` 文件
2. 输入用户名登录（或直接使用默认设置）
3. 选择练习模式开始学习

### 键盘操作

- 直接输入对应字符的五笔编码
- 使用 `Enter` 键确认输入
- 使用 `Esc` 键退出当前界面

### 界面导航

| 界面 | 说明 |
|------|------|
| 登录页 | 输入用户名开始学习 |
| 首页 | 查看概览和进入各功能模块 |
| 选关页 | 选择练习难度等级 |
| 练习页 | 字符/文章输入练习 |
| 结果页 | 查看本次练习成绩 |
| 词书页 | 管理自定义练习词书 |
| 字典页 | 查询五笔编码 |

## 项目结构

```
atomicwubi/
├── index.html          # 主应用文件
├── sounds/             # 音效文件目录
│   ├── correct.mp3     # 正确输入音效
│   └── wrong.mp3      # 错误输入音效
└── *.md                # 开发文档
```

## 音效文件

如需使用完整的音效系统，请添加以下音效文件到 `sounds/` 目录：

- `correct.mp3` - 正确输入时的提示音
- `wrong.mp3` - 错误输入时的提示音

可从免费音效网站获取或自行录制。
