# 音效文件指南

## 需要的音效文件

请在 `sounds/` 目录下创建或下载以下音效文件：

| 文件名 | 用途 | 建议类型 | 建议长度 |
|--------|------|----------|----------|
| `correct.mp3` | 正确输入音效 | 短促的成功音效 | 0.1-0.2秒 |
| `wrong.mp3` | 错误输入音效 | 短促的失败音效 | 0.1-0.2秒 |
| `combo3.mp3` | 3连击音效 | 轻微的上升音效 | 0.2-0.3秒 |
| `combo5.mp3` | 5连击音效 | 中等强度音效 | 0.3-0.4秒 |
| `combo10.mp3` | 10连击音效 | 强烈的成功音效 | 0.4-0.5秒 |
| `achievement.mp3` | 成就解锁音效 | 喜庆的成就音效 | 0.5-0.8秒 |
| `level_complete.mp3` | 关卡完成音效 | 胜利主题音效 | 0.8-1.2秒 |
| `warning.mp3` | 倒计时警告音效 | 急促的警告音效 | 0.2-0.3秒 |
| `click.mp3` | 按钮点击音效 | 轻微的点击声 | 0.1-0.2秒 |

## 获取音效的途径

### 免费音效网站
1. **Freesound.org** - 免费音效库，需要注册
2. **Zapsplat.com** - 免费音效下载
3. **OpenGameArt.org** - 游戏音效资源
4. **SoundBible.com** - 免费音效
5. **Pixabay.com** - 免费音效和音乐

### 创建自己的音效
使用以下工具创建简单的音效：
- **BFXR** (https://www.bfxr.net/) - 在线音效生成器
- **SFXR** - 经典的游戏音效生成器
- **Chiptone** (https://sfbgames.itch.io/chiptone) - 复古风格音效生成器

## 文件格式要求
- 格式: MP3
- 采样率: 44.1kHz (标准)
- 比特率: 128kbps 或更高
- 文件大小: 尽量控制在 100KB 以内

## 测试建议
1. 将所有音效文件放置在 `sounds/` 目录下
2. 打开 `index.html` 测试音效功能
3. 使用音量滑块调整音量
4. 使用音效开关测试开关功能

## 注意事项
- 确保音效文件命名完全匹配
- 音效长度不宜过长，避免影响游戏体验
- 音量不宜过大，以免打扰用户
- 考虑添加版权信息到音效文件

---

**提示**: 如果没有音效文件，系统会自动忽略音效播放，不会影响游戏功能。
