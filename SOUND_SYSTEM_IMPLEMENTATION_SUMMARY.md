## 🚀 使用说明

### 1. 准备音效文件
1. 创建 `sounds/` 目录（已自动创建）
2. 根据 `SOUNDS_GUIDE.md` 下载或创建所需音效文件
3. 将音效文件放入 `sounds/` 目录

### 2. 功能说明

#### 音效播放触发点
- **正确输入**: 输入正确的五笔编码时播放
- **错误输入**: 输入错误或超长时播放
- **连击音效**: 
  - 3连击: `combo3.mp3`
  - 5连击: `combo5.mp3`
  - 10连击: `combo10.mp3`
- **成就解锁**: 获得新成就时播放
- **关卡完成**: 游戏结束时播放
- **倒计时警告**: 剩余5秒时播放

#### 音效控制
- **开关**: 在设置栏点击"音效"开关控制音效启用/禁用
- **音量**: 使用音量滑块调节音量（0-100%）

### 3. 技术特点

#### 音效管理类
```javascript
class SoundManager {
  play(name) - 播放指定音效
  toggle() - 切换音效开关状态
  setVolume(volume) - 设置音量（0-1）
}
```

## 🔧 故障排除

### 常见问题

1. **音效不播放**
   - 检查音效文件是否存在
   - 检查文件名是否正确
   - 检查音效开关是否开启

2. **音量问题**
   - 调整音量滑块
   - 检查系统音量设置

3. **性能问题**
   - 确保音效文件大小合理（< 100KB）
   - 避免同时播放过多音效

### 恢复方案

如果音效系统出现问题：
1. 删除 `sounds/` 目录下的音效文件
2. 系统会自动忽略音效播放，不影响游戏功能
